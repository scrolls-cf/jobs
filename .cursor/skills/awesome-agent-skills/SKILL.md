# Awesome Agent Skills (catalog)

1. Search **`docs/awesome-agent-skills-catalog-full.md`** for vendor, product, or topic (local mirror of the upstream README).
2. Open **`docs/awesome-agent-skills.md`** for this repo’s Cloudflare-focused shortlist.
3. To install a skill, follow the link from the catalog to [officialskills.sh](https://officialskills.sh/) or the vendor repo, then add or mirror `SKILL.md` under `.cursor/skills/<skill-name>/` or `~/.cursor/skills/`.

Prefer vendor-published skills over ad-hoc prompts when the task matches a listed skill.
